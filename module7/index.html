HTML

HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the World Wide Web. It is not a programming language but uses a system of elements and tags to tell a web browser how to display text, images, and other multimedia, acting as the fundamental "skeleton" or blueprint for any webpage.
